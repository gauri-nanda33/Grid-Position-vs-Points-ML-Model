# Grid Position vs Points ML Model

ML project analyzing whether F1 qualifying position predicts championship points (2021–2025). Pulls live session and standings data via the FastF1 API, then fits a simple linear regression using gradient descent from scratch. Visualized as an interactive Streamlit app.
